# Overview

My config for neovim

# Requirements

* neovim
* fzf
* ripgrep
* fasd
* vimplug
* bat

# Quick Installation

``` bash
cd $HOME; git clone https://github.com/Slasat/configs.git
echo 'source $HOME/configs/nvim/init.vim' >> $HOME/.config/nvim/init.vim
```
